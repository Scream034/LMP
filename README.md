
# 🎵 Lite Music Player (LMP) 

**Легковесный клиент YouTube Music, который не жрёт гигабайт оперативки.**

Официальный YTM на Windows — 600–1500 МБ RAM даже когда просто открыт.  
Этот плеер — **250–350 МБ в среднем**.  
И всё.

## ✨ Особенности

*   🚀 **Экстремальная производительность:** Написан на .NET 10 с использованием Avalonia UI. Весь интерфейс оптимизирован для плавной работы даже на слабых ПК.
*   🎧 **Умный стриминг (Memory-First):** Собственная реализация `MemoryFirstCachingStream` с механизмом Read-ahead. Трек начинает играть мгновенно, кешируясь в оперативную память и параллельно сохраняясь на диск. Вы получаете мгновенный старт воспроизведения и минимальное потребление интернет-трафика.
*   🖼️ **Spotify-inspired UI:** Эстетичный интерфейс, плавными переходами и кастомными контролами.
*   📦 **Кеширование всего:** Изображения, результаты поиска и аудиопотоки кешируются локально, что минимизирует количество запросов к сети и экономит трафик.
*   🔊 **Продвинутый звук:** Движок на базе LibVLC с поддержкой усиления (Gain Control), лимита громкости и плавного переключения треков.
*   🌍 **Локализация:** Полная поддержка русского и английского языков (i18n).
*   📚 **Личная библиотека:** Управление плейлистами, система лайков и история прослушиваний, работающая в офлайн-режиме.
*   **Гибкая настройка:** Возможность тонкой настройки интерфейса, качества звука и сетевого взаимодействия.

---

## 🛠 Стек

*   **UI Framework:** [Avalonia UI](https://avaloniaui.net/) (MVVM, ReactiveUI).
*   **Audio Engine:** [LibVLCSharp](https://github.com/videolan/libvlcsharp) (VLC под капотом).
*   **Image Loading:** AsyncImageLoader с кастомным дисковым кешем.
*   **Dependency Injection:** Microsoft.Extensions.DependencyInjection.

## 🚀 Запуск

**Требования:**
*   `.NET 10 SDK` (проект использует самые свежие возможности платформы).
*   Установленный в системе или поставляемый с приложением `VLC`.
*   Желательно `Windows 10/11`
*   Соединение с интернетом (для стриминга и загрузки метаданных)

```bash
# Клонировать репозиторий
git clone https://github.com/Scream034/LiteYTMusicPlayer.git

# Перейти в папку
cd Lite Music Player (LMP)

# Запустить проект
dotnet run
```
